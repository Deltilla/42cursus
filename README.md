### About 42cursus

This repository contains all the projects completed during the **42cursus**, the core curriculum of the **42** school. The 42cursus is an intensive, self-paced, and peer-to-evaluated computer programming training program, focusing on mastering the **C language** fundamentals, system programming, algorithms, and object-oriented programming with **C++**.

The goal is to teach students how to learn, how to be resourceful, and how to work collaboratively, primarily through practical projects.
