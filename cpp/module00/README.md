# CPP Module 00

## Introduction to C++

### Topics Covered

This module introduces the basics of C++ programming:
- **Namespaces**: Organizing code and avoiding name conflicts
- **Classes**: Defining custom data types with member functions and variables
- **Member functions**: Methods that operate on class data
- **Stdio streams**: Input/output operations using cin and cout
- **Initialization lists**: Efficient member variable initialization
- **Static members**: Class-level variables and functions
- **Const**: Constant values and const-correctness

### Exercises

Each exercise builds upon the previous concepts:
- **ex00**: Megaphone - First C++ program using streams
- **ex01**: My Awesome PhoneBook - Creating a simple phonebook with classes
- **ex02**: The Job Of Your Dreams - Working with more complex class structures

### Key Concepts

- Understanding the difference between C and C++
- Learning basic OOP principles
- Using standard input/output streams
- Creating and using classes
- Understanding scope and namespaces

### Compilation

```bash
c++ -Wall -Wextra -Werror -std=c++98 *.cpp -o program_name
```
