# CPP Module 03

## Inheritance

### Topics Covered

This module introduces inheritance in C++:
- **Inheritance basics**: Creating derived classes from base classes
- **Protected members**: Access control for inheritance
- **Virtual functions**: Enabling polymorphic behavior
- **Multiple inheritance**: Deriving from multiple base classes
- **Diamond problem**: Understanding and resolving inheritance issues

### Exercises

- **ex00**: Aaaaand... OPEN! - Basic inheritance
- **ex01**: Serena, my love! - Extended inheritance hierarchy
- **ex02**: Repetitive work - Multiple objects with inheritance
- **ex03**: Now it's weird! - Diamond inheritance and virtual inheritance

### Key Concepts

- Understanding the "is-a" relationship
- Public, protected, and private inheritance
- Constructor and destructor call order
- Virtual functions and vtables
- Solving the diamond problem with virtual inheritance
