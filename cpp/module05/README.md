# CPP Module 05

## Repetition and Exceptions

### Topics Covered

This module introduces exception handling in C++:
- **Try-catch blocks**: Catching and handling exceptions
- **Throwing exceptions**: Creating and throwing custom exceptions
- **Exception classes**: Defining custom exception types
- **Stack unwinding**: Understanding exception propagation
- **RAII**: Resource Acquisition Is Initialization principle

- **ex00**: Mommy, when I grow up, I want to be a bureaucrat!
- **ex01**: Form up, maggots!
- **ex02**: No, you need form 28B, not 28C...
- **ex03**: At least this beats coffee-making

### Key Concepts

- Understanding exception safety
- Creating custom exception classes
- Proper exception handling patterns
- The role of destructors in exception safety
- Resource management with exceptions
- When to use exceptions vs error codes
